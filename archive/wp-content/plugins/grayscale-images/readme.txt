== Description ==

This plugin converts all images to grayscale and show the colored image on hover.

This plugin is based on the work show in [Streamline.lv](http://www.majas-lapu-izstrade.lv/cross-browser-grayscale-image-example-using-css3-js-v2-0-with-browser-feature-detection-using-modernizr/). 
The IE10 and IE11 browser detection was developed by [Tim Pietrusky](http://timpietrusky.com/).

It supports the next browsers:

* [Google Chrome](https://www.google.com/chrome/).
* [Mozilla Firefox](https://www.mozilla.org/firefox/).
* [Apple Safari](http://www.apple.com/safari/).
* [Microsoft IE 7 and above](http://windows.microsoft.com/internet-explorer/).
* [Microsoft Edge](https://www.microsoft.com/en-us/windows/microsoft-edge).
* [Opera](http://www.opera.com/).

The banner and the icon uses [this image](https://pixabay.com/en/beach-shore-coast-ocean-evening-768421/), released under the [CC0 Public Domain license](https://pixabay.com/en/service/terms/#usage).

== Installation ==

1. You can either use the automatic plugin installer or your FTP program to upload the "grayscale-images" directory to your wp-content/plugins directory. 
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's all folks.
